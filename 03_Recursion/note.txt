🧠 Recursion bujhar 2 ta way

1️⃣ Forward thinking:
Ami ekta operation kore recursion ke dibo,
se oita dekhe bakita kore nibe.

2️⃣ Backtracking thinking:
Se amake ekta value dibe (assume korbo),
tarpor oi value er sathe ami amar operation korbo (eta holo backtracking)

🔁 Program flow concept
Recursion function call er ager part = forward
Recursion function call er porer part = backtracking
